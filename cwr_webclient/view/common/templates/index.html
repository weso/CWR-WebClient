Welcome!

This is a prototype for the CWR web application.

It is still under development, but several steps can be tried and tested. Mainly the CWR validation steps. But pay attention that not all the CWR files may be correctly read yet.

Once the application is fully working it will offer several features, which include among them:

  1. CWR file uploading and storage
  2. CWR file validation and acknowledgement file creation

And also will allow connecting to external modules for other related jobs, such as:

  1. Data matching
  2. Results report and manual edition
